───────────────────────────────────────────────────────────────────
  ✨ ЧТО НОВОГО
───────────────────────────────────────────────────────────────────

1. 💰 АВТОМАТИЧЕСКОЕ ПОЛУЧЕНИЕ БАЛАНСА
   
   Где: Конфигурация → Портфель → "💰 Получить со счета"
   
   Что делает:
   • Подключается к вашему T-Bank счету
   • Получает текущий баланс в рублях
   • Автоматически устанавливает начальный капитал
   • Работает в sandbox и production режимах

   Как использовать:
   1. Укажите токен T-Bank в настройках
   2. Выберите режим (песочница/боевой)
   3. Нажмите "💰 Получить со счета"
   4. Баланс установится автоматически
   5. Сохраните конфигурацию (Ctrl+S)

───────────────────────────────────────────────────────────────────

2. 📊 ПРОВЕРКА ПОДКЛЮЧЕНИЯ К T-BANK
   
   Где: Конфигурация → Торговля → "📊 Проверить подключение"
   
   Что показывает:
   ✓ Режим работы (sandbox/production)
   ✓ ID счета
   ✓ Баланс на счете
   ✓ Количество доступных инструментов
   
   Помогает проверить:
   • Правильность токена
   • Работу подключения
   • Доступность счета

───────────────────────────────────────────────────────────────────

3. ℹ️ СПРАВКА ПО ПОЛУЧЕНИЮ ТОКЕНА
   
   Где: Конфигурация → Торговля → "ℹ️ Как получить"
   
   Пошаговая инструкция:
   1. Перейти на https://www.tbank.ru/invest/settings/api/
   2. Войти в аккаунт
   3. Выпустить токен
   4. Скопировать (показывается ОДИН раз!)
   
   ⚠️ Важно: Храните токен в безопасности!

  🐛 РЕШЕНИЕ ПРОБЛЕМ
───────────────────────────────────────────────────────────────────

❌ "Не указан токен T-Bank"
   
   Решение:
   1. Укажите токен в GUI (Торговля → T-Bank токен)
   2. Или в переменной окружения: TINKOFF_TOKEN=t.ваш_токен
   3. Используйте кнопку "ℹ️ Как получить"

───────────────────────────────────────────────────────────────────

❌ "Баланс нулевой"
   
   Для SANDBOX:
   • Создайте новый счет (account_id: null в конфиге)
   • Счет автоматически пополнится на 1 млн ₽
   
   Для PRODUCTION:
   • Пополните счет через приложение T-Bank
   • Подождите несколько минут

───────────────────────────────────────────────────────────────────

❌ "Ошибка подключения к API"
   
   Проверьте:
   ✓ Правильность токена
   ✓ Интернет-соединение
   ✓ Режим (sandbox/production)
   ✓ Библиотеку: pip install tinkoff-investments
