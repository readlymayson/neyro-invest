Пересоздать окружение:

   python -m venv venv
   venv\Scripts\activate.bat
   pip install -r requirements.txt


Обновить все пакеты:
   
   pip install --upgrade fastapi uvicorn[standard] pydantic websockets


Сохранить текущие версии:
   
   pip freeze > requirements_frozen.txt


Установить из сохраненных версий:
   
   pip install -r requirements_frozen.txt


═════════════════════════════════════════════════════════════════════════
  РЕШЕНИЕ ПРОБЛЕМ
═════════════════════════════════════════════════════════════════════════

❓ Окружение не активируется в PowerShell?
   
   Используйте cmd.exe:
   venv\Scripts\activate.bat
   
   Или разрешите скрипты в PowerShell (один раз):
   Set-ExecutionPolicy RemoteSigned -Scope CurrentUser


❓ pip не найден после активации?
   
   Используйте:
   python -m pip install <package>


❓ Нужно переустановить окружение?
   
   Запустите:
   setup_venv.bat
   
   И выберите "Y" для удаления и пересоздания


❓ Пакеты не устанавливаются?
   
   1. Проверьте интернет
   2. Обновите pip: python -m pip install --upgrade pip
   3. Попробуйте установить по одному
